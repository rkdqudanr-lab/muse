# MUSE 🎭 뮤지컬 추천 AI 가이드

뮤지컬과 뮤지컬 영화를 사랑하는 당신을 위한 AI 추천 챗봇

## ✨ 기능

- 🎭 **취향 분석**: 당신의 취향을 파악해 맞춤 추천
- 🎬 **영화 + 무대**: 뮤지컬 영화와 무대 뮤지컬 모두 소개
- 📋 **관람 리스트**: 보고 싶은 작품을 저장하고 관리
- 🎵 **상세 정보**: 줄거리, 대표곡, 매력 포인트 제공

## 🎪 포함된 작품

### 뮤지컬 영화
- 라라랜드, 위대한 쇼맨
- 레미제라블, 시카고
- 물랑루즈, 맘마미아

### 브로드웨이/웨스트엔드
- 라이온킹, 위키드
- 오페라의 유령, 시카고
- 레미제라블, 맘마미아

### 한국 뮤지컬 (2024-2025)
- 광화문연가, 프랑켄슈타인
- 베르테르, 킹키부츠
- 비틀쥬스, 베어 더 뮤지컬
- 4월은 너의 거짓말, 벤자민 버튼

## 🚀 사용 방법

1. 로그인
2. 취향 질문에 답하기
3. 맞춤 추천 받기
4. 관람 리스트에 저장

## 💡 기술 스택

- React 18
- Claude AI (Haiku 4.5)
- Tailwind CSS
- Vercel
